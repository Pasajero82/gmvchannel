0-14 logros: SIGNAL NOVICE

15-29 logros: DATA RUNNER

30-59 logros: SYSTEM ARCHITECT

60+ logros: GHOST IN THE SHELL

Resultado esperado: /// [SYSTEM ARCHITECT] > MENSAJE ///

3. Sistema de Backup "Data-Link" (Teclas T y Shift+T)
Necesito un sistema de "Pasaportes" para recuperar el progreso si se borran las cookies:

Exportar (Tecla T): Al presionar "T", genera un Token alfanumérico con este formato: GMV-VIDEOS_HIST-LOGROS_TOTALES-NIVEL-CHECKSUM.

Seguridad: El CHECKSUM debe ser una validación simple (ej. la suma de los valores en hexadecimal) para evitar ediciones manuales del código.

Muestra el Token en un cuadro de texto para que el usuario pueda copiarlo.

Importar (Shift + T): Al presionar "Shift + T", abre un cuadro de entrada para pegar un Token:

Validación: Debe empezar por GMV- y el Checksum debe coincidir. Si no, lanza el error: "TOKEN CORRUPTO O INVÁLIDO".

Restauración: Si es válido, descompón el string y sobreescribe los valores históricos en el localStorage (statsData).

Refresco: Ejecuta location.reload() para aplicar los cambios inmediatamente.
